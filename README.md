# LangGraph + Gemini WebSocket Chatbot

A full-stack chatbot project using **LangGraph** and **Gemini 2.5 Flash** with real-time WebSocket communication and session memory.  

## Features
- **Backend**: FastAPI server with WebSocket support.
- **Frontend**: Chat UI with chat bubble, header, send icon, clear chat, and close functionality.
- **LangGraph Integration**: Structured node-based conversation flow.
- **Gemini 2.5 Flash**: AI model for assistant responses.
- **Session Memory**: Tracks `session_id` to remember chat history.
- **JSON Output**: Strict structured JSON responses from the chatbot.
